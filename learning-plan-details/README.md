# 12 周学习计划详解（执行版）

本目录把 `LEARNING_PLAN_12_WEEKS.md` 的每一周拆成可直接执行的步骤、检查点和交付物。

- `week-02-http-json.md`
- `week-03-concurrency-context.md`
- `week-04-refactor-layering.md`
- `week-05-postgresql-basics.md`

- `week-07-authn-authz.md`

- `week-10-testing-system.md`
- `week-11-delivery-deployment.md`
- `week-12-summary-advanced.md`
- `weekly-rhythm-template.md`

## 使用方式

1. 每周开始前先看当周文件中的“详细步骤”。
2. 每完成一步就在“验收清单”里勾选。
3. 每周结束时更新“产出物”并做复盘。
4. 遇到阻塞时，优先按“常见风险与排查”处理。

## 建议执行顺序

按周顺序推进，不建议跳周。如果时间不足，先保留“项目步骤 + 验收步骤”，把“扩展任务”延后。
