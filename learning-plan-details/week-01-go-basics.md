# Week 01：Go 语言基础

## 本周目标

能独立写出函数、结构体、接口、错误处理；理解本项目目录结构。

## 详细步骤

1. 通读 Go 基础语法：变量、切片、map、struct、interface、error。
2. 在本地新建一个 `notes/week01.md`（或你习惯的位置）记录每个概念 1 个最小例子。
3. 用 Go 重写 2 个你熟悉的 JS 小工具函数（如去重、分组、节流中的任意两个）。
4. 跑通项目并阅读入口：`cmd/api/main.go`、`internal/...`。
5. 解释并记录 `cmd` 与 `internal` 的职责。
6. 新增 `GET /ping` 接口并返回 `{"message":"pong"}`。

## 建议实践清单

- 使用 `errors.New` 或 `fmt.Errorf` 返回业务错误。
- 给新接口补充最少 1 个单元测试。
- 保持函数短小，单函数只做一件事。

## 验收清单

- [ ] 能口头解释 `cmd` 与 `internal` 的作用。
- [ ] `GET /ping` 可访问且返回正确。
- [ ] 至少完成 2 个 Go 小工具函数重写。
- [ ] 本周代码通过 `make test`。

## 产出物

- Go 基础语法笔记。
- `GET /ping` 的代码与测试。
- Week 01 复盘记录。

## 常见风险与排查

- 不熟悉指针和值传递：优先画内存示意图，再写最小实验代码。
- 错误处理遗漏：统一在 handler 层映射状态码。
